# 🎓 Student Grade Calculator

A simple web application to calculate students' total marks and display the final grade.

## 🚀 Features

* Calculate total marks out of 500
* Show final grade (Excellent, Very Good, Good, Pass, Poor)
* Clean and simple user interface

## 🛠️ Built With

* HTML
* CSS
* JavaScript

## 📌 How to Use

1. Enter marks for all subjects
2. Click on **"Calculator Marks"**
3. View total marks and grade
